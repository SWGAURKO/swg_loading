# swg_loading

A lightweight FiveM loading screen resource.

## author 'SWG'
## Support

Join the Discord: https://discord.gg/CN7P6Wattc

## Overview

Simple, customizable HTML/CSS/JS loading screen for FiveM.

## Contents

- `client.lua` — Client script
- `fxmanifest.lua` — Resource manifest
- `html/` — Loading screen web files (`index.html`, `config.js`, `assets/`)

## Installation

1. Place the `swg_loading` folder in your server's `resources` directory.
2. Add to your `server.cfg`:

```cfg
ensure swg_loading
```



## Configuration

- Edit [html/config.js](html/config.js#L1) to change settings.
- Modify [html/index.html](html/index.html#L1) and files under [html/assets/](html/assets/) to customize visuals.
